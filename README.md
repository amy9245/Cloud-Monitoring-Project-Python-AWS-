☁️ Cloud Resource Monitoring Tool (Python + AWS)

A Python-based cloud monitoring tool that tracks EC2 CPU usage using AWS CloudWatch.

 Features
- Fetches EC2 CPU utilization
- Alerts on high usage
- Uses AWS CloudWatch metrics
- Built with Python & boto3

 Tech Stack
- Python
- AWS EC2
- AWS CloudWatch
- boto3

 Usage
1. Configure AWS credentials
2. Update EC2 instance ID
3. Run the script

 Learning Outcome
- Cloud monitoring
- Python automation
- AWS API integration
